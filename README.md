**PiHole-Panel is a control panel and real-time statistics for the Pi-hole Adblocker**

Install/Update:
```
sudo apt install python3-gi
```
```
sudo sh ./setup.sh
```

Uninstall:
```
sudo sh ./uninstall.sh
```

Troubleshoot:
```
rm ~/.config/pihole_panel_configs.xml
```
